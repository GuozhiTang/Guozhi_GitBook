# 2.6 Linked List

## 1. Node

```java
class Node {
    int data; // The value of this node
    Node next; // The collection to the next node
    Node (int data) { // Set the value of this node as the input value
        this.data = data;
    }
}
```

* For the first node, it is called `head`.
* For the last node, `lastNode.next = null;`

## 2. Object and Connection

```java
Node firstNode = new Node(1); // Set firstNode as node with value 1
Node secondNode = new Node(2);
firstNode.next = secondNode; // Set secondNode as the next node of firstNode
```

## 3. Double Linked List:

* Node:

```java
class Node {
    int data;
    Node next;
    Node prev;
    Node (int data) {
        this.data = data;
    }
}
```

```java
public class MyLink {
    Node head = null; // 头节点

    /**
     * 链表中的节点，data代表节点的值，next是指向下一个节点的引用
     */
    class Node {
        int data; // 节点的对象，即内容
        Node next; // 节点的引用，指向下一个节点

        public Node (int data) {
            this.data = data;
        }
    }

    /**
     * 向链表中插入数据
     * @param d
     */
    public void addNode (int d) {
        Node newNode = new Node(d); // 实例化一个节点
        if (head == null) {
            head = newNode;
            return;
        }
        Node tmp = head;
        while (tmp.next != null) {
            tmp = tmp.next;
        }
        tmp.next = newNode;
    }

    /**
     *
     * @param index：删除第index个节点
     * @return
     */
    public boolean deleteNode (int index) {
        if (index < 1 || index > length()) {
            return false;
        }
        if (index == 1) {
            head = head.next;
            return true;
        }
        int i = 1;
        Node preNode = head;
        Node curNode = preNode.next;
        while (curNode != null) {
            if (i == index) {
                preNode.next = curNode.next;
                return true;
            }
            preNode = curNode;
            curNode = curNode.next;
            i++;
        }
        return false;
    }

    /**
     *
     * @return 返回节点的长度
     */
    public int length() {
        int length = 0;
        Node tmp = head;
        while (tmp != null) {
            length ++;
            tmp = tmp.next;
        }
        return length;
    }

    /**
     * 打印链表
     */
    public void printList() {
        Node tmp = head;
        while (tmp != null) {
            System.out.print(tmp.data + "-->");
            tmp = tmp.next;
        }
        System.out.println();
    }

    public static void main (String[] args) {

        MyLink l1 = new MyLink();
        MyLink l2 = new MyLink();
        l1.addNode(1);
        l1.addNode(3);
        l1.addNode(4);
        l1.addNode(5);
        l1.addNode(6);

        l2.addNode(1);
        l2.addNode(2);
        l2.addNode(6);
        l2.addNode(7);
        l2.addNode(9);

        l1.printList();
        l2.printList();

        System.out.println("head_data1: " + l1.head.data);
        System.out.println("head_data2: " + l2.head.data);

        l1.head = l1.head.next;
        System.out.println("head_data1: " + l1.head.data);

    }
}
```
