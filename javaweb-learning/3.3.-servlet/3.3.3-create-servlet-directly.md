## 3. Configuration

这样直接创建出来的servlet和3.3.2中手动创建出来的有些许区别：

在servlet类上端会有一句话，能帮助我们完成一些之前由web.xml配置的工作：

```java
@WebServlet(name = "TestServlet")
```

我们需要手动配置一个urlPatterns即可

```java
@WebServlet(name = "TestServlet", urlPatterns = "/test")
```
